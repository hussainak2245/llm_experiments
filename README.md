# llm_experiments

Minimal Quarto workspace for DOE-based analysis of LLM accuracy across GSM8K, LeetCode, and HARDMath.

## Files

- `_quarto.yml` + `index.qmd` + `experiments.qmd`: website and analysis entry points
- `ref.bib`: bibliography references
- `data/datasets/`: dataset links/metadata
- `data/results/`: experiment outputs
- `scripts/helpers.py` and `scripts/helpers.R`: helper functions
- `requirements.txt`: Python dependencies
